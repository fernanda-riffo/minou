# Minou — Premium Cat Products Store

A fully responsive e-commerce frontend project built with HTML and CSS.

## About the Project

Minou is a fictional premium cat products store designed as part of my web development portfolio. The site showcases UI/UX design skills, responsive layouts, and clean frontend code.

## Features

- Responsive design — works on desktop, tablet and mobile
- 4 pages: Home, Shop, Product detail and Contact
- CSS custom properties for consistent theming
- Google Fonts integration (Cormorant Garamond + DM Sans)
- Animated preloader
- CSS Grid and Flexbox layouts

## Note

This is a frontend-only project. Cart, login and purchase functionality are not implemented — this is a design and layout showcase.

## Built With

- HTML5
- CSS3
- Google Fonts
- Animate.css
